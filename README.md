# GHCUC EXE - Delta Roblox Executor

Um executor estilo **Delta** para Roblox com interface moderna e recursos avançados.

## 🎮 Características

- ✨ Script executor com console
- 🎯 Injector de scripts Lua
- 📚 Biblioteca de funções Roblox
- 🖥️ Interface gráfica intuitiva
- ⚡ Execução rápida e eficiente
- 🔐 Segurança integrada
